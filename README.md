# Neovim Config 

## Install 
```
# copys old nvim config into your homedirectory
mv ~/.config/nvim/ ~/
# makes nvim directory
mkdir ~/.config/nvim
# clones the repo
git clone https://github.com/LamaV10/lamaVim ~/.config/nvim
```
